## install ##

From rubygems:  
~~~
    [sudo] gem install num4regana
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4regana.git  
    cd num4regana  
    gem build *.gemspec
    gem install *.gem
~~~
